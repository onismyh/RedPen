# Recipe examples

These files are starter scaffolds for common RedPen workflows.

- `proofread.json` — grammar and clarity fixes
- `tighten.json` — concise executive phrasing
- `reviewer.json` — collaborative reviewer mode with comment-style reasons

Use them as templates, then run:

```bash
redpen apply examples/sample.docx @examples/recipes/proofread.json -o revised.docx
```
